# Zeno Bit-Engine & NP-Monkey Search

A dual-engine exploration into the intersection of discrete computation, classical physics, and NP stochastic search. Built as a proof-of-concept for the limits of continuous mathematics when mapped to a Boolean lattice.

## Concept 1: The NP-Monkey Engine (`index.html`)
A high-performance stochastic string solver. It bypasses the exponential time complexity of the "Infinite Monkey Theorem" by applying a dynamic fitness function (Hamming Distance) to a parallel array of random walkers, effectively forcing a wave-function collapse onto a target string (e.g., the NIV Bible).

## Concept 2: The Zeno Gravity Engine (`zeno.html`)
A visual translation of Newtonian calculus (y = h - ½gt²) into raw machine code logic. It demonstrates how standard physics requires infinite division (FPU calculation), while discrete reality relies on integer carry-chains (ALU bit-shifting) that eventually hit a hard Zeno limit at the Least Significant Bit.

**Architecture Files:**
* `monkey-engine.js` - Real-time buffer allocation and persistent offline processing.
* `engine.js` - The discrete falling logic.
* `kernel.c` - The "Assembly of Reality" underlying the bitwise logic gates.
